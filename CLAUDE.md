# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 저장소 목적

Anthropic 공식 강좌(https://anthropic.skilljar.com/) 17개를 수강하며 실습 코드와 결과물을 관리하는 학습 저장소다. 행정 사무관 업무(보고서 작성, 뉴스기사 분석)에 Claude Code를 활용하는 것이 최종 목표다.

## Git 설정

- 원격 저장소: `git@github.com:seenbeomlee/practice_claude.git`
- SSH 키: `~/.ssh/seenbeomlee-GitHub` (SSH config에 `Host github.com`으로 등록됨)
- 기본 브랜치: `main`

강좌는 7일 로드맵으로 구성되며, 일차별로 디렉토리를 만들어 실습 파일을 관리한다.
